# Gorodulin Tap

## How do I install these formulae?

`brew install gorodulin/tap/<formula>`

Or `brew tap gorodulin/tap` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "gorodulin/tap"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
